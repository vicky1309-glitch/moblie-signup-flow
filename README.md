# SoundWave — Music App Signup Flow

A simple, animated authentication UI for a music streaming app, built with plain HTML, CSS, and JavaScript in a red and black theme.

## Overview

This project recreates a complete mobile signup and login flow inside a realistic phone-shaped frame, including a welcome screen, sign up, email verification, login, forgot password, OTP verification, password reset, and success screens, all connected with smooth slide transitions.

## Features

- Realistic phone mockup frame with a status bar and notch
- Welcome screen with an animated spinning vinyl record and audio equalizer bars
- Sign up screen with live password strength meter and form validation
- Email verification screen with a 6-digit auto-advancing code input, countdown timer, and resend option
- Login screen with show/hide password toggle and remember me option
- Forgot password flow with email entry, OTP verification, and new password screen
- Animated success screens for both email verification and password reset
- Fully responsive single-file design, no external libraries required

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, transitions)
- Vanilla JavaScript (no frameworks or dependencies)

## How to Run

1. Clone or download this repository.
2. Open `music_signup_flow.html` directly in any modern web browser.
3. No build steps, installs, or servers are required.

## Customization

All colors, fonts, and spacing are controlled through CSS variables at the top of the stylesheet, so the theme can be changed easily by editing the `:root` block.
